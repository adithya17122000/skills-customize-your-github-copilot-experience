# Python Data Visualization

## 🎯 Objective

Use Python plotting libraries such as `matplotlib` or `plotly` to build clear and engaging charts that reveal trends in data.

## 📝 Tasks

### 🛠️ Task 1: Create a visual chart

#### Description
Use the sample dataset in `starter-code.py` to build a visual chart with either `matplotlib` or `plotly`.

#### Requirements
Completed program should:

- load or define the sample data in Python
- create at least one chart type (line, bar, scatter, or pie)
- include a title, axis labels, and a legend or data labels

### 🛠️ Task 2: Customize and explain your visualization

#### Description
Add a second chart or enhance your first chart with styling and insights.

#### Requirements
Completed program should:

- produce at least two visual elements or chart types
- customize the chart colors, line style, or markers
- include a short text summary of what the chart shows
- save the output to a file or display it clearly when run
